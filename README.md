## 🧠 Overview

Learning Web3 is hard.  
Testing understanding is even harder.

**OG Dev Quiz** is an interactive platform designed to help users **learn, test, and reinforce their knowledge** of Web3 concepts and the OpenGradient ecosystem.

It turns passive learning into **active engagement**.

---

## 🚧 The Problem

- Learning resources are often passive  
- No clear way to test real understanding  
- Builders struggle to measure progress  
- Engagement drops without interaction  

**Knowledge without testing is incomplete.**

---

## 💡 The Solution

OG Dev Quiz provides:

- 🎯 Interactive quizzes for active learning  
- 🧠 Knowledge validation for builders  
- ⚡ Fast, engaging question flow  
- 📊 Immediate feedback on performance  

Users don’t just learn —  
they **prove what they understand**.

---

## ⚙️ How It Works

```text
User Starts Quiz → Answer Questions → Instant Evaluation → Score & Feedback
````

A simple loop that reinforces learning through interaction.
